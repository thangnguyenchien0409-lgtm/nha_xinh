js
useEffect(() => {

👉 Mỗi lần bạn bấm "Xem thêm", limit tăng lên (ví dụ từ 16 → 24).

Nhưng còn 1 vấn đề nữa

Reducer của bạn (fetchGetAllProduct.fulfilled) có đang ghi đè products mỗi lần fetch không?
Nếu ghi đè thì khi đổi page, dữ liệu sẽ chỉ hiện sản phẩm của page mới → không giống "xem thêm".

👉 Trường hợp muốn load thêm nối vào cuối, reducer phải làm như sau:

js
extraReducers: (builder) => {
  builder.addCase(fetchGetAllProduct.fulfilled, (state, action) => {
    if (action.meta.arg.page === 1) {
      // Nếu là page 1 thì reset
      state.products = action.payload.products;
    } else {
      // Nếu page > 1 thì nối thêm
      state.products = [...state.products, ...action.payload.products];
    }
  });
};

👉 Tóm lại, để bấm "Xem thêm" mà đổi trang đúng:

  1. useEffect phải phụ thuộc page.

  2. Reducer phải nối thêm dữ liệu khi page > 1.
